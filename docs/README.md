jump to whichever doc matches what you're trying to do.

The `superpowers/` subfolder contains historical hackathon planning
artefacts (specs and plans) — they describe earlier UI iterations and
are kept for context, not as current source of truth. When the docs
above disagree with anything in `superpowers/`, the docs above win.
